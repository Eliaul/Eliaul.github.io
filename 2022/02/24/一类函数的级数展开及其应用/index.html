一类函数的级数展开及其应用

发表于 2022-02-24 更新于 2022-02-27 分类于 复变函数与积分变换 阅读次数： Valine：
本文字数： 18k 阅读时长 ≈ 36 分钟

本文着重讨论以下两个函数及其相关函数的性质及其展开式，并给出一些积分和级数的例子

\[\displaystyle{\frac{1}{\cosh x+\cos x},\frac{1}{\cosh x-\cos x}}\]

1.\(f\left( x \right) =\dfrac{1}{\cosh x+\cos x}\)

由于

\[\displaystyle{\cos x+\cosh x=\cos x+\cos \left( \mathrm{i}x \right) =2\cos \left( \frac{1+\mathrm{i}}{2}x \right) \cos \left( \frac{1-\mathrm{i}}{2}x \right) }\]

所以其在复数域的全部零点为\(x=\dfrac{\left( 2n+1 \right) e^{\pm \pi \mathrm{i}/4}\pi}{\sqrt{2}}\),其中\(n\)为所有整数。那么根据\(\cos\,(x)\)的无穷乘积展开，可得到\(f(x)\)的无穷乘积展开为：

\[\begin{align*}f\left( x \right) &=\frac{1}{2\prod\limits_{n=1}^{\infty}{\left( 1-\frac{2\mathrm{i}x^2}{\left( 2n-1 \right) ^2\pi ^2} \right) \left( 1+\frac{2\mathrm{i}x^2}{\left( 2n-1 \right) ^2\pi ^2} \right)}} \\ &=\frac{1}{2\prod\limits_{n=1}^{\infty}{\left( 1+\frac{4x^4}{\left( 2n-1 \right) ^4\pi ^4} \right)}}\\&=\frac{1}{2}\prod_{n=1}^{\infty}{\frac{\left( 2n-1 \right) ^4\pi ^4}{\left( 2n-1 \right) ^4\pi ^4+4x^4}}\end{align*}\]

取对数求导可得到\(\dfrac{\sinh x-\sin x}{\cosh x+\cos x}\)的有理分式展开：

\[\displaystyle{\frac{\sinh x-\sin x}{\cosh x+\cos x}=\sum_{n=1}^{\infty}{\frac{16x^3}{\left( 2n-1 \right) ^4\pi ^4+4x^4}}}\]

根据Mittag-Leffler定理，\(f(x)\)也可以展开成有理分式。考虑函数：

\[\displaystyle{g\left( z \right) =\frac{\pi z}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \cos \left( \pi z \right) \cosh \left( \pi z \right)}}\]

显然其在复平面内有两个一阶极点\(z=\dfrac{\pm \left( 1+\mathrm{i} \right) x}{2\pi}\)，且\(g(z)\)在这两点的留数均为

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =\frac{1}{\pi \left( \cosh x+\cos x \right)}}\]

那么构造一个大圆弧围道，很容易验证\(g(z)\)在围道上半径趋近于无穷大的时候的积分为0。那么：

\[\displaystyle{\frac{2}{\pi \left( \cosh x+\cos x \right)}+\sum_{n=-\infty}^{\infty}{\mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2} \right]}+\sum_{n=-\infty}^{\infty}{\mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2} \mathrm{i} \right]}=0}\]

也就是

\[\begin{align*}&\frac{2}{\pi \left( \cosh x+\cos x \right)} \\ =&-\sum_{n=-\infty}^{\infty}{\left[ \frac{\left( -1 \right) ^{n-1}}{\cosh \left( \frac{2n+1}{2}\pi \right)}\left( \frac{2\left( 2n+1 \right)}{\left( 2n+1 \right) ^2\pi ^2-2\mathrm{i}x^2}+\frac{2\left( 2n+1 \right)}{\left( 2n+1 \right) ^2\pi ^2+2\mathrm{i}x^2} \right) \right]} \\ =&\sum_{n=-\infty}^{\infty}{\frac{\left( -1 \right) ^n}{\cosh \left( \frac{2n+1}{2}\pi \right)}\frac{4\left( 2n+1 \right) ^3\pi ^2}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ =&\sum_{n=0}^{\infty}{\frac{\left( -1 \right) ^n}{\cosh \left( \frac{2n+1}{2}\pi \right)}\frac{8\left( 2n+1 \right) ^3\pi ^2}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}\end{align*}\]

即

\[\displaystyle{\frac{1}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{\left( -1 \right) ^n}{\cosh \left( \frac{2n+1}{2}\pi \right)}\frac{4\left( 2n+1 \right) ^3\pi ^3}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

类似的，选取不同的\(g(z)\)可以得到不同分式的有理分式展开：

如果选取

\[\displaystyle{g\left( z \right) =\frac{\pi \sin \left( \frac{2\pi}{1+\mathrm{i}}z \right)}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \cos \left( \pi z \right) \cosh \left( \pi z \right)}}\]

那么可以发现：

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,z=\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =\frac{\left( 1-\mathrm{i} \right) \sin x}{x\left( \cosh x+\cos x \right)}}\]

于是：

\[\begin{align*}&\frac{2\left( 1-\mathrm{i} \right) \sin x}{x\left( \cosh x+\cos x \right)} \\ =&-\sum_{n=-\infty}^{\infty}{\left\{ \mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2} \right] +\mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2}\mathrm{i} \right] \right\}} \\ =&-\sum_{n=-\infty}^{\infty}{\left( -\frac{4}{\left( 2n+1 \right) ^2\pi ^2-2\mathrm{i}x^2}+\frac{4\mathrm{i}}{\left( 2n+1 \right) ^2\pi ^2+2\mathrm{i}x^2} \right)} \\ =&\left( 1-\mathrm{i} \right) \sum_{n=-\infty}^{\infty}{\frac{4\left( \left( 2n+1 \right) ^2\pi ^2-2x^2 \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}\end{align*}\]

即

\[\displaystyle{\frac{\sin x}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{4x\left( \left( 2n+1 \right) ^2\pi ^2-2x^2 \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

选取

\[\displaystyle{g\left( z \right) =\frac{\pi z\cos \left( \frac{2\pi}{1+\mathrm{i}}z \right)}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \cos \left( \pi z \right) \cosh \left( \pi z \right)}-\frac{\pi z}{\pi ^2z^2-\frac{\mathrm{i}x^2}{2}}}\]

那么

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,z=\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =\frac{\cos x}{\pi \left( \cosh x+\cos x \right)}-\frac{1}{2\pi}}\]

故

\[\begin{align*}&\frac{2\cos x}{\pi \left( \cosh x+\cos x \right)}-\frac{1}{\pi} \\ =&-\sum_{n=-\infty}^{\infty}{\left\{ \mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2} \right] +\mathrm{Res}\left[ g\left( z \right) ,z=\frac{2n+1}{2}\mathrm{i} \right] \right\}} \\ =&-\sum_{n=-\infty}^{\infty}{\left( -\frac{2\mathrm{i}\left( 2n+1 \right) \tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^2\pi ^2-2\mathrm{i}x^2}+\frac{2\mathrm{i}\left( 2n+1 \right) \tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^2\pi ^2+2\mathrm{i}x^2} \right)} \\ =&-\sum_{n=-\infty}^{\infty}{\frac{8\left( 2n+1 \right) x^2\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}\end{align*}\]

即

\[\displaystyle{\frac{\cos x}{\cosh x+\cos x}=\frac{1}{2}-\sum_{n=0}^{\infty}{\frac{8\pi \left( 2n+1 \right) x^2\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

在上述式子中用\(\mathrm{i}x\)替代\(x\)，则

\[\begin{align*}&\frac{\sinh x}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{4x\left( \left( 2n+1 \right) ^2\pi ^2+2x^2 \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\cosh x}{\cosh x+\cos x}=\frac{1}{2}+\sum_{n=0}^{\infty}{\frac{8\pi \left( 2n+1 \right) x^2\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}\end{align*}\]

注意到

\[\begin{align*}&\frac{\sin x}{\cosh x+\cos x} \\ =&\frac{2e^x\sin x}{e^{2x}+2e^x\cos x+1}=\frac{2e^x\sin x}{\left( e^x+\cos x \right) ^2+\sin ^2x} \\ =&\frac{2e^x\sin x}{\left( e^x+e^{\mathrm{i}x} \right) \left( e^x+e^{-\mathrm{i}x} \right)}=\mathrm{i}e^x\left( \frac{1}{e^x+e^{\mathrm{i}x}}-\frac{1}{e^x+e^{-\mathrm{i}x}} \right) \\ =&\mathrm{i}\left( \frac{1}{1+e^{\left( \mathrm{i}-1 \right) x}}-\frac{1}{1+e^{-\left( \mathrm{i}+1 \right) x}} \right) =\mathrm{i}\sum_{n=0}^{\infty}{\left( -1 \right) ^n\left[ e^{\mathrm{i}nx-nx}-e^{-\mathrm{i}nx-nx} \right]} \\ =&-2\sum_{n=0}^{\infty}{\left( -1 \right) ^ne^{-nx}\sin nx}=2\sum_{n=1}^{\infty}{\left( -1 \right) ^{n-1}e^{-nx}\sin nx}\,\,(x>0)\end{align*}\]

类似的，有

\[\displaystyle{\frac{e^x+\cos x}{\cosh x+\cos x}=2\sum_{n=0}^{\infty}{\left( -1 \right) ^ne^{-nx}\cos nx}\,\,\left( x>0 \right) }\]

于是我们得到了下列展开式：

\[\begin{align*}&\frac{1}{\cosh x+\cos x}=\frac{1}{2}\prod_{n=1}^{\infty}{\frac{\left( 2n-1 \right) ^4\pi ^4}{\left( 2n-1 \right) ^4\pi ^4+4x^4}} \\ &\frac{1}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{\left( -1 \right) ^n}{\cosh \left( \frac{2n+1}{2}\pi \right)}\frac{4\left( 2n+1 \right) ^3\pi ^3}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\sin x}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{4x\left( \left( 2n+1 \right) ^2\pi ^2-2x^2 \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\cos x}{\cosh x+\cos x}=\frac{1}{2}-\sum_{n=0}^{\infty}{\frac{8\pi \left( 2n+1 \right) x^2\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\sinh x}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{4x\left( \left( 2n+1 \right) ^2\pi ^2+2x^2 \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\cosh x}{\cosh x+\cos x}=\frac{1}{2}+\sum_{n=0}^{\infty}{\frac{8\pi \left( 2n+1 \right) x^2\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}} \\ &\frac{\sin x}{\cosh x+\cos x}=2\sum_{n=1}^{\infty}{\left( -1 \right) ^{n-1}e^{-nx}\sin nx}\,\,\left( x>0 \right) \\ &\frac{e^x+\cos x}{\cosh x+\cos x}=2\sum_{n=0}^{\infty}{\left( -1 \right) ^ne^{-nx}\cos nx}\,\,\left( x>0 \right) \end{align*}\]

2.\(f\left( x \right) =\dfrac{1}{\cosh x-\cos x}\)

由于

$x-x=( x ) -x=2( x ) ( x ) $

那么根据\(\sin\,(x)\)的无穷乘积，可得到

\[\displaystyle{\frac{1}{\cosh x-\cos x}=\frac{1}{x^2}\prod_{n=1}^{\infty}{\frac{4n^4\pi ^4}{4n^4\pi ^4+x^4}}}\]

取对数求导可得

\[\displaystyle{\frac{\sinh x+\sin x}{\cosh x-\cos x}=\sum_{n=1}^{\infty}{\frac{4x^3}{4n^4\pi ^4+x^4}}+\frac{2}{x}}\]

考虑

\[\displaystyle{g\left( z \right) =\frac{\pi z}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \sin \left( \pi z \right) \sinh \left( \pi z \right)}}\]

那么

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =-\frac{\mathrm{i}}{\pi \left( \cosh x-\cos x \right)}}\]

所以

\[\begin{align*}&-\frac{2\mathrm{i}}{\pi \left( \cosh x-\cos x \right)} \\ =&-\sum_{n\in \mathbb{Z} \backslash\left\{ 0 \right\}}{\left[ \frac{\left( -1 \right) ^n2n}{\sinh \left( n\pi \right)}\frac{1}{2n^2\pi ^2-\mathrm{i}x^2}-\frac{\left( -1 \right) ^n2n}{\sinh \left( n\pi \right)}\frac{1}{2n^2\pi ^2+\mathrm{i}x^2} \right]}-\frac{2\mathrm{i}}{\pi x^2} \\ =&-\sum_{n\in \mathbb{Z} \backslash\left\{ 0 \right\}}{\frac{\left( -1 \right) ^n2n}{\sinh \left( n\pi \right)}\frac{2\mathrm{i}x^2}{4n^4\pi ^4+x^4}}-\frac{2\mathrm{i}}{\pi x^2}\end{align*}\]

即

\[\displaystyle{\frac{1}{\cosh x-\cos x}=\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^n4n}{\sinh \left( n\pi \right)}\frac{\pi x^2}{4n^4\pi ^4+x^4}}}\]

考虑

\[\displaystyle{g\left( z \right) =\frac{\pi \sin \left( \frac{2\pi}{1+\mathrm{i}}z \right)}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \sin \left( \pi z \right) \sinh \left( \pi z \right)}}\]

则

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =\frac{-\left( 1+\mathrm{i} \right) \sin x}{x\left( \cosh x-\cos x \right)}}\]

故

\[\begin{align*}&\frac{-2\left( 1+\mathrm{i} \right) \sin x}{x\left( \cosh x-\cos x \right)} \\ =&-\sum_{n\in \mathbb{Z} \backslash \left\{ 0 \right\}}{\left[ -\frac{2\mathrm{i}}{2n^2\pi ^2-\mathrm{i}x^2}-\frac{2}{2n^2\pi ^2+\mathrm{i}x^2} \right]}-\frac{2\left( 1+\mathrm{i} \right)}{x^2} \\ =&\left( 1+\mathrm{i} \right) \sum_{n\in \mathbb{Z} \backslash \left\{ 0 \right\}}{\frac{4n^2\pi ^2-2x^2}{4n^4\pi ^4+x^4}}-\frac{2\left( 1+\mathrm{i} \right)}{x^2}\end{align*}\]

即

\[\displaystyle{\frac{\sin x}{\cosh x-\cos x}=\frac{1}{x}-\sum_{n=1}^{\infty}{\frac{4n^2\pi ^2x-2x^3}{4n^4\pi ^4+x^4}}}\]

考虑

\[\displaystyle{g\left( z \right) =\frac{\pi z\cos \left( \frac{2\pi}{1+\mathrm{i}}x \right)}{\left( \pi ^2z^2-\frac{\mathrm{i}x^2}{2} \right) \sin \left( \pi z \right) \sinh \left( \pi z \right)}-\frac{\pi z}{\pi ^2z^2-\frac{\mathrm{i}x^2}{2}}}\]

那么

\[\displaystyle{\mathrm{Res}\left[ g\left( z \right) ,\frac{\pm \left( 1+\mathrm{i} \right) x}{2\pi} \right] =-\frac{\mathrm{i}\cos x}{\pi \left( \cosh x-\cos x \right)}-\frac{1}{2\pi}}\]

于是

\[\begin{align*}&\frac{-2\mathrm{i}\cos x}{\pi \left( \cosh x-\cos x \right)}-\frac{1}{\pi} \\ =&-\sum_{n\in \mathbb{Z} \backslash \left\{ 0 \right\}}{\left[ \frac{2n\coth \left( n\pi \right)}{2n^2\pi ^2-\mathrm{i}x^2}-\frac{2n\coth \left( n\pi \right)}{2n^2\pi ^2+\mathrm{i}x^2} \right]}-\frac{2\mathrm{i}}{\pi x^2} \\ =&-\sum_{n\in \mathbb{Z} \backslash \left\{ 0 \right\}}{\frac{4\mathrm{i}nx^2\coth \left( n\pi \right)}{4n^4\pi ^4+x^4}}-\frac{2\mathrm{i}}{\pi x^2}\end{align*}\]

即

\[\displaystyle{\frac{\cos x}{\cosh x-\cos x}=-\frac{1}{2}+\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{4n\pi x^2\coth \left( n\pi \right)}{4n^4\pi ^4+x^4}}}\]

用\(\mathrm{i}x\)替代\(x\)，得

\[\begin{align*}&\frac{\sinh x}{\cosh x-\cos x}=\frac{1}{x}+\sum_{n=1}^{\infty}{\frac{4n^2\pi ^2x+2x^3}{4n^4\pi ^4+x^4}} \\ &\frac{\cosh x}{\cosh x-\cos x}=\frac{1}{2}+\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{4n\pi x^2\coth \left( n\pi \right)}{4n^4\pi ^4+x^4}}\end{align*}\]

注意到

\[\begin{align*}&\frac{\sin x}{\cosh x-\cos x} \\ =&\frac{2e^x\sin x}{e^{2x}-2e^x\cos x+1}=\frac{2e^x\sin x}{\left( e^x-\cos x \right) ^2+\sin ^2x} \\ =&\frac{2e^x\sin x}{\left( e^x-e^{\mathrm{i}x} \right) \left( e^x-e^{-\mathrm{i}x} \right)}=-\mathrm{i}e^x\left( \frac{1}{e^x-e^{\mathrm{i}x}}-\frac{1}{e^x-e^{-\mathrm{i}x}} \right) \\ =&-\mathrm{i}\left( \frac{1}{1-e^{\left( \mathrm{i}-1 \right) x}}-\frac{1}{1-e^{-\left( \mathrm{i}+1 \right) x}} \right) =-\mathrm{i}\sum_{n=0}^{\infty}{\left[ e^{\mathrm{i}nx-nx}-e^{-\mathrm{i}nx-nx} \right]} \\ =&2\sum_{n=1}^{\infty}{e^{-nx}\sin nx}\,\,\left( x>0 \right) \end{align*}\]

类似的

\[\displaystyle{\frac{e^x-\cos x}{\cosh x-\cos x}=2\sum_{n=0}^{\infty}{e^{-nx}\cos nx}\,\,\left( x>0 \right) }\]

于是我们得到了下列展开式：

\[\begin{align*}&\frac{1}{\cosh x-\cos x}=\frac{1}{x^2}\prod_{n=1}^{\infty}{\frac{4n^4\pi ^4}{4n^4\pi ^4+x^4}} \\ &\frac{1}{\cosh x-\cos x}=\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^nn}{\sinh \left( n\pi \right)}\frac{4\pi x^2}{4n^4\pi ^4+x^4}} \\ &\frac{\sin x}{\cosh x-\cos x}=\frac{1}{x}-\sum_{n=1}^{\infty}{\frac{4n^2\pi ^2x-2x^3}{4n^4\pi ^4+x^4}} \\ &\frac{\cos x}{\cosh x-\cos x}=-\frac{1}{2}+\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{4n\pi x^2\coth \left( n\pi \right)}{4n^4\pi ^4+x^4}} \\ &\frac{\sinh x}{\cosh x-\cos x}=\frac{1}{x}+\sum_{n=1}^{\infty}{\frac{4n^2\pi ^2x+2x^3}{4n^4\pi ^4+x^4}} \\ &\frac{\cosh x}{\cosh x-\cos x}=\frac{1}{2}+\frac{1}{x^2}+\sum_{n=1}^{\infty}{\frac{4n\pi x^2\coth \left( n\pi \right)}{4n^4\pi ^4+x^4}} \\ &\frac{\sin x}{\cosh x-\cos x}=2\sum_{n=1}^{\infty}{e^{-nx}\sin nx}\,\,\left( x>0 \right) \\ &\frac{e^x-\cos x}{\cosh x-\cos x}=2\sum_{n=0}^{\infty}{e^{-nx}\cos nx}\,\,\left( x>0 \right) \end{align*}\]

上述展开式可以构造很多积分级数。

在

\[\displaystyle{\frac{1}{\cosh x+\cos x}=\sum_{n=0}^{\infty}{\frac{\left( -1 \right) ^n}{\cosh \left( \frac{2n+1}{2}\pi \right)}\frac{4\left( 2n+1 \right) ^3\pi ^3}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

中令\(x=0\)可得

\[\displaystyle{\sum_{n=0}^{\infty}{\frac{\left( -1 \right) ^n}{\left( 2n+1 \right) \cosh \left( \frac{2n+1}{2}\pi \right)}}=\frac{\pi}{8}}\]

在

\[\displaystyle{\frac{1}{x^2}\left( \frac{\cos x}{\cosh x+\cos x}-\frac{1}{2} \right) =-\sum_{n=0}^{\infty}{\frac{8\pi \left( 2n+1 \right) \tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

中令\(x\rightarrow 0\)可得

\[\displaystyle{\sum_{n=0}^{\infty}{\frac{\tanh \left( \frac{2n+1}{2}\pi \right)}{\left( 2n+1 \right) ^3}}=\frac{\pi ^3}{32}}\]

在

\[\displaystyle{\frac{\sinh x-\sin x}{\cosh x+\cos x}\frac{1}{x^3}=\sum_{n=0}^{\infty}{\frac{16}{\left( 2n+1 \right) ^4\pi ^4+4x^4}}}\]

中取\(x=\pi\)可得

\[\displaystyle{\sum_{n=0}^{\infty}{\frac{1}{\left( 2n+1 \right) ^4+4}}=\frac{\pi \coth \left( \frac{\pi}{2} \right)}{16}}\]

在

\[\displaystyle{\left( \frac{1}{\cosh x-\cos x}-\frac{1}{x^2} \right) \frac{1}{x^2}=\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^n}{\sinh \left( n\pi \right)}\frac{4\pi n}{4n^4\pi ^4+x^4}}}\]

中令\(x\rightarrow 0\)可得

\[\displaystyle{\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^n}{n^3\sinh \left( n\pi \right)}}=-\frac{\pi ^3}{360}}\]

在

\[\displaystyle{\left( \frac{\cos x}{\cosh x-\cos x}+\frac{1}{2}-\frac{1}{x^2} \right) \frac{1}{x^2}=\sum_{n=1}^{\infty}{\frac{4n\pi \coth \left( n\pi \right)}{4n^4\pi ^4+x^4}}}\]

中令\(x\rightarrow 0\)可得

\[\displaystyle{\sum_{n=1}^{\infty}{\frac{\coth \left( n\pi \right)}{n^3}}=\frac{7\pi ^3}{180}}\]

在

\[\displaystyle{\left( \frac{\sinh x+\sin x}{\cosh x-\cos x}-\frac{2}{x} \right) \frac{1}{x^3}=\sum_{n=1}^{\infty}{\frac{4}{4n^4\pi ^4+x^4}}}\]

中取\(x=\pi\)可得

\[\displaystyle{\sum_{n=1}^{\infty}{\frac{1}{4n^4+1}}=\frac{\pi}{4}\tanh \left( \frac{\pi}{2} \right) -\frac{1}{2}}\]

积分部分：

\[\begin{align*}&\int_0^{\infty}{\frac{\sin x}{\cosh x+\cos x}}\mathrm{d}x \\ =&2\sum_{n=1}^{\infty}{\left( -1 \right) ^{n-1}\int_0^{\infty}{e^{-nx}\sin nx}\mathrm{d}x} \\ =&\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^{n-1}}{n}}=\ln 2\end{align*}\]

\[\begin{align*}&\int_0^{\infty}{\left( \frac{e^x+\cos x}{\cosh x+\cos x}-2 \right)}\mathrm{d}x \\ =&2\sum_{n=1}^{\infty}{\left( -1 \right) ^n\int_0^{\infty}{e^{-nx}\cos nx}\mathrm{d}x} \\ =&-\ln 2\end{align*}\]

\[\begin{align*}&\int_0^{\infty}{\frac{x\sin x}{\cosh x+\cos x}}\mathrm{d}x \\ =&2\sum_{n=1}^{\infty}{\left( -1 \right) ^{n-1}\int_0^{\infty}{xe^{-nx}\sin nx}\mathrm{d}x} \\ =&\sum_{n=1}^{\infty}{\frac{\left( -1 \right) ^{n-1}}{n^2}}=\frac{\pi ^2}{12}\end{align*}\]
